# WordFlow 单词复习

一个手机端优先的单词卡复习小应用。当前版本无需安装依赖，直接打开 `index.html` 就能使用。

## 功能

- 每天批量输入单词和释义
- 学习 / 复习两种模式
- 20 / 50 / 100 三个档位
- 单词卡正反面翻页动画
- `记住了` / `不熟` 记录学习状态
- 词库面板查看所有已添加单词
- 可用于添加到手机桌面的 app 图标
- 使用浏览器 `localStorage` 保存词库

## 输入格式

每行一个单词，推荐写法：

```text
apple 苹果
renew / 更新、恢复
capture: 捕获、拍摄
```

## 使用方式

1. 双击打开 `index.html`。
2. 输入当天要背的单词，点击 `加入今天计划`。
3. 选择 `学习` 或 `复习`。
4. 选择 `20`、`50` 或 `100`。
5. 点击 `开始`，点击卡片翻面，按熟悉程度选择 `记住了` 或 `不熟`。
6. 点击底部 `词库`，查看所有已添加的单词、释义和学习状态。
